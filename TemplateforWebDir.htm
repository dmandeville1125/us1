It is meant to be the main page of the research web directory.

The 1 in 1a.htm means Volume 1, the a in 1a.htm means first page or “page a”.

The .htm in 1a.htm is the file extension and it means the file is in HTML format.

HTML format is used in web pages. So it is meant for the web or internet.

The 1a.htm file will be referred to quite often by the lower files!

This topic page example is from something I am working on now. There’s quite a bit going on but I try to keep topic pages as simple as possible. There can only be five links to webpages per topic page. This is so things can be added and added and added but everything is chopped up by 5. I can only really go up to 7 pages or about 35 webpage links over all. That’s because of how I have the “Pages a b c d e f g” going on and only so many letters can be put in there before it gets to be too much.

So, there’s “(Reference files available here)” link at the bottom of the page. This is a bit of a back door to where the real work is. So, the topic file above is named “1a2a1a.htm” for first volume, second section, and first topic. The letters, a, a, a, are pages so pages are lettered starting with a and then b and then c. Pages should only contain 5 items max. This is to keep visual appearance as pleasant as possible. But it means lots and lots of small files! Volumes only have 5 things then the number is increased. Volume 1 only has four sections, though, because the introduction is the first section. Volume 2 will start with section 6 and go to section 10. Volume 3 will go 11-15. Volume 4 will go 16-20.
